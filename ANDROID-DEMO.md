## 🛠️ **الطريقة اليدوية:**

### **المتطلبات:**
- ✅ Node.js (v16 أو أحدث)
- ✅ حساب Expo مجاني
- ✅ اتصال إنترنت مستقر

### **الخطوات:**

#### **1. تثبيت EAS CLI:**
```bash
npm install -g @expo/eas-cli
```

#### **2. تسجيل الدخول:**
```bash
eas login
```

#### **3. تكوين المشروع:**
```bash
eas build:configure
```

#### **4. بناء النسخة التجريبية:**
```bash
eas build --platform android --profile demo
```

## 📱 **طرق التوزيع:**

### **1. QR Code (الأسهل):**
- امسح الكود بكاميرا الهاتف
- سيفتح رابط التحميل
- اضغط "تحميل" و"تثبيت"

### **2. رابط مباشر:**
- انسخ الرابط وأرسله
- افتح على الهاتف
- حمل وثبت التطبيق

### **3. ملف APK:**
- حمل الملف على الكمبيوتر
- انقله للهاتف عبر USB/البلوتوث
- ثبت من مدير الملفات
