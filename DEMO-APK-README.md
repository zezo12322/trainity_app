## 🔧 **إعدادات الأمان:**

### **تفعيل مصادر غير معروفة:**

#### **Android 8+:**
1. **الإعدادات** → **التطبيقات والإشعارات**
2. **الوصول الخاص للتطبيقات**
3. **تثبيت تطبيقات غير معروفة**
4. اختر **المتصفح** أو **مدير الملفات**
5. فعل **السماح من هذا المصدر**

#### **Android 7-:**
1. **الإعدادات** → **الأمان**
2. فعل **مصادر غير معروفة**
